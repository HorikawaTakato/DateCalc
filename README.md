# 日数計算機 DateCalc

入力した日付と今日の差分を計算するツールです。  
紀元前（例：紀元前100年）から西暦9999年まで対応しています。

CLIとWebブラウザの2つの使い方があります。

---

## ファイル構成

```
DateCalc/
├── DateCalc.py          # 計算ロジック + CLIインターフェース
├── DateCalc_server.py   # Flask Webサーバー（API）
├── DateCalc.html        # Webフロントエンド
└── requirements.txt     # 依存パッケージ
```

---

## 動作環境

- Python 3.10 以上
- Flask（Web版のみ）

---

## セットアップ

```bash
# リポジトリをクローン
git clone https://github.com/HorikawaTakato/DateCalc.git
cd DateCalc

# 依存パッケージをインストール（Web版を使う場合）
pip install -r requirements.txt
```

---

## 使い方

### CLI版

```bash
python DateCalc.py
```

起動後、`年 月 日` の順に半角スペース区切りで入力します。

```
日数計算機
今日 — 2025年5月22日(木)
========================================
コマンド: 日付入力（例: 2030 3 15）/ h=履歴 / c=履歴クリア / q=終了

日付を入力（年 月 日）> 2030 1 1
  2030年1月1日(火)
  1685日後 / 4年224日後

日付を入力（年 月 日）> -100 3 15
  紀元前100年3月15日(日)
  ...
```

| コマンド | 動作 |
|---|---|
| `年 月 日`（例：`2030 1 1`） | 差分を計算 |
| `h` | 履歴を表示（最大8件） |
| `c` | 履歴をクリア |
| `q` | 終了 |

### Web版

```bash
python DateCalc_server.py
```

起動後、ブラウザで `http://localhost:5000` を開きます。  
`Ctrl+C` でサーバーを停止できます。

---

## 主な機能

- 指定日と今日の差分を日数で表示（365日以上は年数も併記）
- 紀元前の日付に対応（例：`-100 3 15` = 紀元前100年3月15日）
- 曜日の自動計算
- 計算履歴の管理（最大8件、重複は自動排除）
